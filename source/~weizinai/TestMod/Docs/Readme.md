# 功能

## 矿井地图修改

可以在`GenericModConfigMenu`中设置想要访问的矿井地图的ID.用于测试1.6中新添加的矿井地图。

## 火山地图修改

可以在`GenericModConfigMenu`中设置想要访问的火山地图的ID.用于测试1.6中新添加的火山地图.

## 鱼的狂潮修改

修改了鱼的狂潮事件发生的概率.用于测试1.6中新添加的鱼的狂潮事件.

## 钓鱼金色宝箱修改

修改了钓鱼金色宝箱的概率.用于测试1.6中新添加的钓鱼金色宝箱.
