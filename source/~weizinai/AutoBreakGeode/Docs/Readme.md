# 介绍

该模组让你能够自动的砸开晶球(包括金色椰子，谜之盒，金色谜之盒等)。你只需要手持晶球，然后按下快捷键(默认是F，可以在`config.json`或`GMCM`中修改)
，该模组就会开始工作。或者你也可以在`config.json`或`GMCM`中将`绘制开始按钮`选项设为`true`，这样在晶球面板的左上角就会有一个开始按钮，手持晶球点击该按钮也可以开始工作。

# 功能

- 自动砸开晶球
- 砸开晶球快速动画
