# 介绍

该模组能让你修改好感度衰减的数值.如果你将它设置为0,那么好感度将不会衰减,就像`Friend Forever`模组一样.当然该模组比`Friend Forever`模组更加强大,它还可以修改礼物,垃圾桶的好感度衰减数值.

# 功能

- 每日对话修改
- 礼物修改
- 垃圾桶修改
- 抚摸动物修改
- 喂食动物修改
