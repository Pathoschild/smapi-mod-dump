# 介绍

该模组为小屋添加了更多的功能。

# 功能

- `拜访小屋信息`。当你拜访一个小屋时，游戏右下角会显示一条与小屋主人相关的信息。
- `小屋主人名字标签`。在小屋上会显示小屋主人的名字。
- `总在线时间标签`。在小屋上会显示小屋主人的总在线时间。
- `上次在线时间标签`。在小屋上会显示小屋主人的上次在线时间。
- `小屋面板`
  - 客机玩家可以移动小屋和修改小屋外观
  - 主机玩家可以移动小屋、拆除小屋和建造小屋
- `重置小屋`。你可以不删除小屋，只删除小屋主人与小屋的联系。
