[size=6][b]Self Serve[/b][/size]

Don't want to wait for shop owners like Marnie and Clint while they're gone? You no longer have to! Access any shop services even while its owner isn't attending it.

[img]https://staticdelivery.nexusmods.com/mods/1303/images/headers/20985_1711335150.jpg[/img]
[size=5][b]Prerequisites[/b][/size]

Mandatory and optional requirements for use of this mod.

- [url=https://www.nexusmods.com/stardewvalley/mods/2400]SMAPI 4.0[/url] (required)
- [url=https://www.nexusmods.com/stardewvalley/mods/5098]GMCM 1.12[/url] (optional, but highly recommended)

[b][size=5]Installation[/size][/b]

1. Download and install the required prerequisites.
2. Download the latest version of this mod through [url=https://www.nexusmods.com/stardewvalley/mods/20985?tab=files]NexusMods[/url] or [url=https://github.com/Dunc4nNT/StardewMods/releases]GitHub releases[/url].
3. Unzip the downloaded file into the [u]Stardew Valley/Mods[/u] directory.
4. Launch the game using SMAPI.

[size=5][b]Configuration[/b][/size]

The mod's configurable options can be found below. These can either be adjusted through [url=https://www.nexusmods.com/stardewvalley/mods/5098]Generic Mod Configuration Menu (GMCM)[/url] or the [u]config.json[/u] file. Editing the options using GMCM is recommended. GMCM will also provide descriptive tooltips if the option itself is not clear.

[list]
[*]Pierre's General Store
[*]Willy's Fish Shop
[*]Ice Cream Shop
[*]Blacksmith
[*]Carpenter's
[*]Marnie's Animal Shop
[*]Hospital
[*]Stardrop Saloon Bar
[*]Bookseller's Shop
[*]Traveling Merchant
[*]Resort Bar
[*]Sandy's Oasis
[/list]
[b][size=5]How to Use[/size][/b]

Once you've configured the mod the way you want, simply go to a shop and interact with it!

[b][size=5]Compatibility[/size][/b]

This mod is made for Stardew Valley 1.6 using SMAPI 4.0. This mod will [b]not[/b] work on versions prior to those mentioned.
