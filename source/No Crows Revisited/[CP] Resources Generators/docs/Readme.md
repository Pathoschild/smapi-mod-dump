# [CP] Resources Generators


![screenshots1.png](screenshots1.png)

## Description

Craftable machines to generate basic resources. Including clay, wood, stone, fiber , etc.

## Machine Types

### ClayMachine

1. 100% Clay
2. 005% Coal
3. 003% Geode
4. 001% Omni Geode

### FiberMachine

1. 100% Fiber
2. 005% Mixed Seeds

### StoneMachine

1. 100% Stone
2. 005% Copper Ore
3. 004% Iron Ore
4. 003% Gold Ore
5. 002% Iridium Ore

### WoodMachine

1. 100% Wood
2. 010% Hardwood

## Shop modification

All Machines available at Clint Shop.
