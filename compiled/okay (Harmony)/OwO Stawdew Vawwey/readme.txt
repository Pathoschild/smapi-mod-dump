Install:
1. Make sure you've installed and set up SMAPI first https://smapi.io
2. Unzip the contents of this folder into Stardew Valley/Mods
3. Launch the game with SMAPI.

Usage:
In the SMAPI console, type in "help" and look under OwO Stawdew Vawwey.
Commands:
owomods - Changes which modifications are made to text
oworandom - Randomly enable and disable this mod during gameplay.
owoshuffle - Reshuffles random elements, giving a different result on text.

Source code can be found at: https://github.com/CrunchyDuck/OwO-Stawdew-Vawwey
Nexus page: https://www.nexusmods.com/stardewvalley/mods/8923
