Stardew Valley mod Random Start Day
https://www.nexusmods.com/stardewvalley/mods/13225

- "UseWinter28toYear1" in config: Decides if it return to year 1 when starts Spring 1st. Don't set as false unless you got error or freezing when Winter 28 to Spring 1.

* Note: Do not change seasons or dates through SMAPI console. This command recalculates the number of days played starting from Spring 1, which may change your gameplay experience in unintended ways.
please use instead "rsd_set_date".
