NPC Name: #NAME#

This is an experimental very raw method to edit NPC dialogues and provides no error checking.
These edits will not erase the original game text and will only be displayed if Stardew Web is installed. If you do mess it up, save an empty text box and the normal text will be restored.
For more details on how dialogues work, check out the Wiki.
Edits you make will immediately be available in the game, no need to restart.
Where is this going? The end version of this should be able to save your edits into a Mod file that your can share.
For now if you want to share your customizations with a friend, they need to be running Stardew Web and you can send them the file 'customstring.json' from 'Mod Directory\StardewWeb' and they need to place the file in their 'Mod Directory\StardewWeb' directory.
