Frequently Asked Questions
  • Can I access the web page from another PC? - Yes
    To view the web site on another machine there are 3 steps:
    1. Change the 'Server Address' parameter to "http://+:8080/" in the Config view of the mod or in the config.json file in the mod's directory. The game will have to be re-started for the change to take effect.
    2. Open a command window, as administrator, and run the following command:

      netsh http add urlacl url=http://+:8080/ user=Everyone

      This command allows the mod to use the 8080 port, without having to run as administrator. The alternative to this command is run stardew as administrator, which is not a good way to run.
    3. Run windows defender firewall and add a program exception for stardewmoddingapi. (The rule is most likely already there.)

    You will now be able to view the mod from any machine on your local network.

    DO NOT PORT FORWARD TO THIS ADDRESS.

    This mod is not hardened to be exposed to the nasty internet.
