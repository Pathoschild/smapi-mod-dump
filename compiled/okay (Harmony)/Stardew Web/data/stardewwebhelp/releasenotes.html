Release Notes
  • 0.7.6
    • Added support for Dynamic Game Assets (DGA)
      • DGA items are searchable and spawnable.
      • DGA items are supported in Chest Manager
      • In the Stardew Search enter 'dga' to list all DGA assets.
      • Some items are shown that are not meant to be spawnable and may result in interesting items if spawned
      • DGA item images are not currently available in Stardew 1.5.5 (coming)
    • Enhanced Producers Framework support
      • Added additional data mining to extract more rule definitions
    • Added more Configuration Issue logging.
      • Configuration issues listed by mod.
    • Added configurable wiki lookup from stardewvalleywiki.com embeded in Search results.
    • Testing Features
      • Added the ability to Warp to any mine level
      • Added the ability to launch Events
        • Drill into an event and press 'Start Event' to start the Event
      • Toggle Rain
      • Mature All Crops
      • Activate Sprinklers
      • Pet Animals
    • Added new Monster Eradications view.
    • Updated code base to match the latest Stardew 1.5.5 changes
    • Added support for Custom Farming Redux
      • Machine images are not displayed, yet.
      • Machines can be spawned from 'Stardew Search'
      • Machines are shown under 'Big Craftables' in the Catalogue
    • Added more details to the Modder Utils -> Maps or Characters -> Locations
      • Shows all Warps for the location
      • List all NPCs in the location
      • List all Animals in the location
      • List of all Buildings in the location
    • Status Bar updates in real time (Configurable)
    • Started transitioning layout to bootstrap framework to support mobile clients.
      • Some pages may have taken a step backwards in their look.
    • 0.7.5
      • Mainly to align code with 64 bit beta release.
      • Think-n-Talk web module added - allows you to create Think or Talk speech bubbles above your Farmer's head
    • 0.7.2 - Second Beta Release
      • Did some HTML tricks to make image loading less intensive.
      • Moved the screen shot routine to the backgroud to not impact location switches.
      • Released 'Chest Manager' view to allow the moving of Items to and from Chests.
      • Released 'Planning' view to display details of in season crops.
      • Re-organized menu tabs to clean up the navigation bar.
      • Fixed various bugs occurring while content packs were being parsed.
      • Added start of customizable background image.
      • Made maximum return hits in Stardew Web Search configurable, for those with large Mod collections.
      • Tweaked the image fetching routines to ensure a tree is a tree and a fruit is a fruit.
      • Released 'Invincible' mode to help in Monster battles.
      • Changed background image from hard coded to configurable.
      • Added the ability to see or un-see Events.
      • Added the ability to view Letters and mark them as read or unread.
      • Tweaking of the Search Engine to increase its accurary.
      • Moved all start of Day tasks to the backgroud, to not impact game play.
      • Upgraded News Report to include:
        • A check to see if the Queen of Sauce Recipe is new.
        • Reports on any Tools being upgraded and when they will be ready.
        • Added News Items for Salmonberry and Blackberry season
      • Added tab to view and create animated Gif images from the screenshots from Daily Screenshot.
    • 0.6.1 - Initial Release
      • When pages with a large number of images load for the first time, some images may not load. Doing a page refresh fixes the issue for the rest of the session.
      • Use Warps at your own risk. You could end up in some interesting spots. (Try warping to a Fish Tank)
