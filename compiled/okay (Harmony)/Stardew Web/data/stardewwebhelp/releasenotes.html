Release Notes
  • 1.0.0 (Stardew Valley 1.6 Release)
    • Expanded Stardew Realty support:
      • Added seasonal override info to map grid.
      • Enhanced Expansion Customize abilities
    • Added Today's Movie to news on the Home page
    • Added What's Playing at the theater.
    • Added list of place visited for Farmer details.
    • Added Warp Home configuration for warping indoors or outdoors
    • Enhanced In Season Crops to include profit details
    • Added Game Data Editor to allow the editing of individual Game Data fields Supported types
      • Crops
      • Farm Animals
      • Objects
    • Added "Fishing Journal" - lists all fish caught, the record size and the largest size.
    • Added Junimo Hut information to the (Farm -> Machines)
    • Added "From"" info to Mail listings.(Farmer -> Mailbox)
    • Enhanced around Town Report to include beach forage. (Home -> Around Town)
    • Added 'Vegas Wedding' testing option. (Cheats -> Farmer)
    • Touched up error handling.
    • Enhanced DGA Integration
    • Updated code to SAMPI 4 compatible.
    • Added NPC Tracker View - lists current location of all NPCs with the option to warp too the NPC, if warping is enabled. (Characters -> Tracking)
    • Added Custom location warping to the Warp Studio view (requires Warping to be enabled).
      Support for:
      • Railroad Station
      • Custom Aquarium
      • East Scarp
      • Ridgeside Village
    • Added GMCM configuration support for web server and few other options
    • Added Automation Tasks.
      • Turn on all Torches at a set time
      • Turn off all Torches at a set time
      • Pet all animals
      • Remove weeds on Mondays
      • Talk to known NPCs
    • Added screen zoom in and out buttons to the Remote Control
    • Added the ability to set NPC friendship level (For testing)
    • Added Animals added details to BFAV mods in Mod details
  • 0.8.4
    • Updated code to SMAPI 3.15
    • Added "Building Manager" for Animal Houses.
      • Shows details of house occupants.
      • Does the house have a heater.
      • Does the house have an Auto Grabber.
      • Does the Coop have an incubator.
      • Door status with toogle.
      • Hay quantity. (Can be topped up)
      • Warp to the house. (If warps are enabled)
    • Added Macro langauge
      • Use for setting up testing scenarios
      • Variety of commands for clearing things and adding things
        • Add buildings
        • Add objects to buildings
        • Add animals to Animal Houses
        • Add Chests Anywhere Category and Name to Chest
        • Add crops or flowers
        • Add pathing and fencing
      • Crop writing command to draw lettering formed with crops and flowers as the palette
    • Added "Crop Writer" functionality for writing letters and numbers or images to be written in flowers or crops.
    • Add NPC family details, using NPC Genealogy mod.
    • Added support for Animals Need Water
      • Animals needing water are shown in the Farm Task Report
    • Added the ability to complete individual Community Centre Bundles.
    • Fixed a bug in Livestock page when the Farm has Fish Ponds.
    • Fixed the Daily Screenshot Gif creator, it is functional again.
    • Fixed a bug with preserved items (Honey, Wine, Aged Roe, Juice and Pickles) not showing an image in the Inventory views.
    • Fixed a bug spawning Weapons, Rings and Clothing. All work correclty now.
    • Fixed issue with trying to load pages before the internal library had finished loading.
    • Fixed a bug in Location detail loading.
    • Fixed a bug where Chest Manager would throw errors when moving items, mainly when it was fighting with Automate.
    • Fixed bugs where dragging and dropping caused items to disappear in the Chest Manager.
    • Added more detailed views of Livestock.
    • Added a "Remote Control" (farvourites) button to the Status bar.
    • Added quality details to machine outputs in Search View.
    • Added "Farm Report" view to display a summary report of the Farm.
    • Modified "Add Animal" functionality to add the selected Aniaml to the current Animalhouse the Farmer is in.
    • Added indicator to "In Season Crops" to show which crops it is too late in the season to plant.
    • Added mine monster predictions to the "Around Town" page.
    • Added the option to integrate (NSFW) Valley Girls events into Daily News.
    • Added the ability to "Cook" recipes.
    • Added the ability to Harvest crops (By crop type).
  • 0.8.3
    • A few codes fixes and cleaning to be in synch with final Stardew Valley 1.5.5
    • Fixed up DGA support, DGA images now displayed in 1.5.5
    • Modified the Chest Manager HTML to, hopefully, make it usable. The quality of an Item hangs down in the GUI, to be fixed
    • Added Train notifications. If enabled, the next coming train will be displayed in the Status bar. Clicking on it will warp you to the Train.
    • Included a Train time fix, can be disabled, that fixes incorrect train times that normally would be skipped.
    • Many HTML changes that for the most part are not noticeable. This is to make the display mobile friendly.
    • Added an Tab (Farm -> Utilities) to show the status of objects in the Utility Grid mod and its related mods.
  • 0.7.6
    • Added support for Dynamic Game Assets (DGA)
      • DGA items are searchable and spawnable.
      • DGA items are supported in Chest Manager
      • In the Stardew Search enter 'dga' to list all DGA assets.
      • Some items are shown that are not meant to be spawnable and may result in interesting items if spawned
      • DGA item images are not currently available in Stardew 1.5.5 (coming)
    • Enhanced Producers Framework support
      • Added additional data mining to extract more rule definitions
    • Added more Configuration Issue logging.
      • Configuration issues listed by mod.
    • Added configurable wiki lookup from stardewvalleywiki.com embeded in Search results.
    • Testing Features
      • Added the ability to Warp to any mine level
      • Added the ability to launch Events
        • Drill into an event and press 'Start Event' to start the Event
      • Toggle Rain
      • Mature All Crops
      • Activate Sprinklers
      • Pet Animals
    • Added new Monster Eradications view.
    • Updated code base to match the latest Stardew 1.5.5 changes
    • Added support for Custom Farming Redux
      • Machine images are not displayed, yet.
      • Machines can be spawned from 'Stardew Search'
      • Machines are shown under 'Big Craftables' in the Catalogue
    • Added more details to the Modder Utils -> Maps or Characters -> Locations
      • Shows all Warps for the location
      • List all NPCs in the location
      • List all Animals in the location
      • List of all Buildings in the location
    • Status Bar updates in real time (Configurable)
    • Started transitioning layout to bootstrap framework to support mobile clients.
      • Some pages may have taken a step backwards in their look.
    • 0.7.5
      • Mainly to align code with 64 bit beta release.
