Game Macros
Creating a macro file:
  1. Create a text file in the StardewWeb\data\macros directory.
  2. Edit the text file adding the desired macro commands.
  3. Save the file.
  4. Start the Game.
  5. Go to the Uyilities -> Macros view
  6. In the Macro Console box select the file created in Step #1 and press 'Run'.
Macro Commands:
  • activatesprinklers:
    Activates all spinklers.
  • addpath:
    Fills the rectangle with pathing tiles.
  • addobject:
    Adds an object to the current location.

    addobject objectId x,y

  • build:
    Builds the specified building at the supplied location.

    build BuildingName x,y [AnimalType AnimalQty]

  • buildingadd:
    Adds an object to the current building.
  • clearall:
    Clears everything off the current map.
  • fillsilos:
    Fills all Silos to the maximum.
  • junimopack:
    Creates a growing area with sprinklers,scarecrows, pathing and a central junimo hut.
  • grass:
    Plants grass starter in the defined rectangle.
  • location:
    Sets the action location to supplied name.
  • petall:
    Pets all animals.
  • verbose:
    Toggles verbose logging.
  • warpto:
    Warps the Farmer to the specified location.
