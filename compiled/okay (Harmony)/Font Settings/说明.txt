简介：
    本模组旨在让玩家能够设置游戏字体。


基本使用：
    - 字体界面位于游戏菜单从左到右最后一栏（画着两个T的那栏）。
    - 游戏内大致分为三种字体：小字体（SmallFont）、中字体（DialogueFont）、对话字体（SpriteText）。（如你所见，中文翻译和英文是不搭的，因为中文是我根据实际情况取名的，请留心）
      
    - 界面分上下两部分，上面是一个简单的示例系统，下面是设置界面。
        - 示例系统：更直观地反映情况。示例文本可以修改，见下“修改配置”段。同时提供一些按键更改示例的显示方式：
            - 合并示例：     将两个示例文本重合。
            - 显示边框：     显示每个字的边框。
            - 显示文字：     显示示例的文字。
            - 右上角的移动按钮，用以微调偏移量。点击后，出现横竖两个滑动条，调整整体的偏移；再次点击，关闭。(极值可配置)

        - 设置界面：点击左右箭头以切换你要设置的字体，每种字体都有如下选项：
            - 启用字体：     勾上表示启用自定义的字体；不勾表示使用原版字体。
            - 字体：         设置字体款式。
            - 大小：         设置字体大小，单位为像素。(极值可配置)
            - 字间距：       设置每个字之间的距离。(极值可配置)
            - 行间距：       设置每一行之间的距离。(极值可配置)

    - 请确保两个示例文本差不多，这样就设置好了。注意，不要差太多，否则一些内容可能因为太大太小太胖太瘦显示不出来。
    - 最后，点击OK，设置完成。


修改配置：
    - ExampleText       示例文本（其中\n表示换行，建议两行，即\n一次，这样方便设置行间距）
    - MaxFontSize       字体大小 最大值。（默认100）
    - MinSpacing        字间距 最小值。（默认-10）
    - MaxSpacing        字间距 最大值。（默认10）
    - MaxLineSpacing    行间距 最大值。(默认100)
    - MinCharOffsetX    横轴偏移量 最小值。（默认-10）
    - MaxCharOffsetX    横轴偏移量 最大值。（默认10）
    - MinCharOffsetY    纵轴偏移量 最小值。（默认-10）
    - MaxCharOffsetY    纵轴偏移量 最大值。（默认10）

兼容性：
    当前版本（0.4.0）
        兼容Smapi 3.14.0及以上，Windows、MacOS、Linux系统，
        语言兼容所有游戏内置语言（语言选择菜单初始3x4=12种语言）。但是使用拉丁字符的语言（en、pt、es、de、fr、it、tr、hu）无法设置对话字体，因为游戏是使用另一种方法——精灵表，来绘制字符的。
        字体格式兼容TrueType（.ttf, .ttc）、OpenType（.otf, .otc, .ttf, .ttc）。

        For latin language (en, pt, es, de, fr, it, tr, hu) users, 
        it's hard to custom dialogue font because game draws those texts with a spritesheet, and its logic in code is really hardcoded to patch.

其他问题：
    问：这个模组支持联机吗？一定要所有玩家同时安装吗？
    答：支持；不需要，实际上对联机毫无影响。
