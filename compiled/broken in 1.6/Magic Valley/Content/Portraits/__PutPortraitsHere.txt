Direct put  "npcName.png" / "npcName_Beach.png" / "npcName/npcName_season.png" (or *.xnb)  in this folder.

Or put them in inner folder, and set config file:

Create file "_PortraitsFolders.json" in this dir. The content format is as follows:

["Folder1","Folder2","Folder3/internalFolder","FolderN"]

Replace Folder* with Actual portraits folder name.
The portraits will be displayed in order.(Show the previous directory portraits first)
