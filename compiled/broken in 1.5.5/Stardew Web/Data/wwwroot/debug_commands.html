#MENUBAR#
apt Brings up an in game menu to preview character animations.
animalinfoDisplays the animal count in an in game popup.
buildBuilds the specified building on the Farm.
buildcoop x yBuild a coop at the specified location
characterinfoDisplays the number of characters in the current map.
deletearchClears museum, fish caught, minerals and mail.
doesitemexist xSearches to see if x is in inventory anywhere.
everythingshopOpens in game menu with everything for sale.
specialsSets all items in your wallet.
clearspecialsClears all items in your wallet.
localInfoDescribes the current location.
netlogToggles multiplayer logging.
netclearClears the multiplayer log.
netdumpDumps the contents of the multiplayer log to the console.
logbandwidthToggle multiplayer bandwidth logging.
movieSends player to the movies with random NPCs.
nethostStarts the multiplayer server.
netjoinBrings up the join game menu.
pmTurns the in game display into pan mode.
exitExits pan display mode.
getallquestsAdds all quests to the players journal.
pausetimePauses the in game time.
phoneBrings up the in game phone call menu.
pppOutputs the palyer's position to the console.
pgbOutputs the locations of the 4 gem birds.
pstmWarps your spouse to your location.
removebuildingsRemoves all buildings from the Farm.
runmacroRun debug commands from the specified text file.
sb xWill put text bubble above NPC x.
setupfarmClears the Farm and sets up a new one.
sleepPuts the player to sleep.
spreaddirtPlaces dirt on the Farm.
spreadseeds xSpreads seeds x on the all hoed land on the Farm.
trlt Brings up an in game menu to preview tailoring things.
warpshopWarps you to the shop of the specified NPC.
wctWarps specified NPC to specified location.
wctmWarps specificed NPC to you.
wtcWarps you to the specified NPC.
wtpWarps you to another player.
setupfishpondfarmCreates farm full of fish ponds
spawncoopsandbarns xSpawns x number of coops and barns
setupbigfarmCreates a big farm.
