Introduction

Stardew Access is a Stardew Valley mod that focuses on making the game blind accessible for windows, linux and mac. The mod uses the Tolk library on windows, libspeechdwrapper on linux and libspeak on mac for interacting with the active screen reader. The mod adds features like read tile, tile viewer, warnings, etc., to help the player gather information regarding the sorrounding and the player. It also patches the various menus and the fishing mini-game (more mini-games will be patched soon) to make them accessible.
