Install:
1. Make sure you've installed and set up SMAPI first https://smapi.io
2. Unzip the contents of this folder into Stardew Valley/Mods
3. Launch the game with SMAPI.

In game, text scrolling will be immediately set to the last character in the sentence.

Source code can be found at: https://github.com/CrunchyDuck/Stardew-Skip-Text
Nexus page: https://www.nexusmods.com/stardewvalley/mods/8947
