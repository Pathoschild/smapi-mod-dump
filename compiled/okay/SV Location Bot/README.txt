Using instructions:
﻿1. Invite bot to your server.
2. Run !init command on server. This command must be used only once by server admin.
﻿3. Run !signup <login> <password> in bot's private messages.
4. Run !set_guild command on the server where you are going to play.
5. Unpack mod in Mods folder in the game directory.
6. Set your login and password in "info.ini" file, located in mod directory.
7. Join to any channel in Discord.
8. Run game with SMAPI.
