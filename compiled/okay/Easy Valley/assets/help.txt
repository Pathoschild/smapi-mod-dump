Customized items sold by Villager's Personal Shop: (Right click the villager's name in the "FriendshipData" page, then can choose Villager's Personal Shop)
	Create file "custom_npc_personal_shops.json" in MOD's "assets" directory. Edit and modify the content according to the corresponding format and save it. (format reference "npc_personal_shops.json" file)
	The field "npcName" corresponds to the name of villager;
	The field "ItemId" corresponds to the QualifiedItemId of Item; Unpacking Game File Wiki Instructions: https://stardewvalleywiki.com/Modding:Editing_XNB_files#xnbcli ;  see Item Id: (Content/Data/Objects.json) (Content/Data/BigCraftables.json)
	The field "Price" corresponds to the price of Item;

中文说明：
自定义村民私人商店出售的物品：（在“村民友谊信息”界面右键点击村民名称，可以选择打开村民的私人商店）
	在MOD的“assets”目录下，创建“custom_npc_personal_shops.json”文件，并将“npc_personal_shops.json”文件内容复制过来，按对应格式编辑修改内容后保存即可。
	字段“npcName”值对应村民的英文名称；
	字段“ItemId”值对应物品代码中的"QualifiedItemId"；物品Id需要查询Wiki及解包游戏文件(https://stardewvalleywiki.com/Modding:Editing_XNB_files#xnbcli) (Content/Data/Objects.json) (Content/Data/BigCraftables.json)
	字段“Price”值对应商店中物品的售价；
