Release Notes
  • 0.7.0 - Second Beta Release
    • Did some HTML tricks to make image loading less intensive.
    • Moved the screen shot routine to the backgroud to not impact location switches.
    • Released 'Chest Manager' view to allow the moving of Items to and from Chests.
    • Released 'Planning' view to display details of in season crops.
    • Re-organized menu tabs to clean up the navigation bar.
    • Fixed various bugs occurring while content packs were being parsed.
    • Added start of customizable background image.
    • Made maximum return hits in Stardew Web Search configurable, for those with large Mod collections.
    • Tweaked the image fetching routines to ensure a tree is a tree and a fruit is a fruit.
    • Released 'Invincible' mode to help in Monster battles.
    • Changed background image from hard coded to configurable.
    • Added the ability to see or un-see Events.
    • Added the ability to view Letters and mark them as read or unread.
    • Tweaking of the Search Engine to increase its accurary.
    • Moved all start of Day tasks to the backgroud, to not impact game play.
    • Upgraded News Report to include:
      • A check to see if the Queen of Sauce Recipe is new.
      • Reports on any Tools being upgraded and when they will be ready.
      • Added News Items for Salmonberry and Blackberry season
    • Added tab to view and create animated Gif images from the screenshots from Daily Screenshot.
  • 0.6.1 - Initial Release
    • When pages with a large number of images load for the first time, some images may not load. Doing a page refresh fixes the issue for the rest of the session.
    • Use Warps at your own risk. You could end up in some interesting spots. (Try warping to a Fish Tank)
