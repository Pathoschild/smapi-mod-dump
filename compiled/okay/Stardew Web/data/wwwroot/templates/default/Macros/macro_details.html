Quick Reference
activatesprinklersActivates all sprinklers in the current location.
addobject objectId x,y [chestcat] [chestname] [remove TF]Adds the object ojbectId to the current location at x,y.
addobjects objectId x,y x1,y1 [chestcat] [chestname] [remove TF]Adds the object ojbectId to the current locations at x,y-x1,y1.
addtf x,y x1,y1 tfType tfIdAdds the Terrain Feature tfId to the current location at x,y. If it is a chest you can specify the Chest Category and Chest Name
addpath x,y x1,y1 typeTypeFills the defined square with tiles of type tileType.
addsign x,y signId detailIdAdds a sign with the detailId object for display.
addstring text x,y bgId strokeId fontname hoedirtWrites text at location x,y using bgId for the background and strokeId for the stroke.
build building x,y animal qtyAdds the building to the current location at x,y and adds qty animals.
buildingadd objectId x,yAdds the object objectId to the last building added at location x,y.
buildingadds objectId x,y x1,y1Adds the objectId to the last building added at location x,y to x1,y1.
buildingaddchest x,y category nameAdds a chest to the last building built at x,y and assigns the ChestsAnywhere names.
clearallRemoves everything from the location.
clearcharactersRemoves all NPCs from the location.
cleardebrisRemoves all Debris from the location.
clearobjectsRemoves all Objects from the location.
clearanimalsRemoves all Animals from the location.
clearlargeRemoves all Large Terrian Features from the location.
clearbuildingsRemoves all Buildings from the location.
fillsilosFills all silos in the current location.
gate x,yadds a gate at x,y.
grass x,y x1,y1Fills the defined square with grass.
hoedirt x,y x1,y1 cropIdHoes the defined square and adds the crop cropId.
ironfence x,yadds an iron fence at x,y
junimopack x,y cropIdAdds a Junimo hut with 8 fields with crop cropId.
location locationSets the macro location context to location.
petallPets all animals in the current location.
set name valueDefines a variable with the supplied value.
verboseShow macro steps in console window.
warpto location x,yWarp the Farmer to location at x,y.
